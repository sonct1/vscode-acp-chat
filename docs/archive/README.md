# Archive

Use this area for superseded, historical, or migrated documents that should remain available for context but are no longer source-of-truth.

Typical files:

- `<date>-<topic>.md`
- `<topic>-archive.md`

Each archived document should explain what replaced it and link to the current source-of-truth when available.

Do not put active product requirements, current API contracts, current runbooks, or active implementation plans here.
